## Features

- [x] firebase setup
- [x] movie catalog UI
- [x] adaptive layout for mobile devices

## How To
### Install Dependencies
```
yarn install
```
### Development
to start development server run:
```
yarn start
```
### Tests
to run unit tests execute:
```
yarn test
```
### Deployment
- create production build:
```
yarn build
```
- deploy to firebase:
```
yarn deploy
```
